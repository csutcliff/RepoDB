# Operations

Complete reference of all RepoDB database operations.

## Query Operations

- **[Query](/docs/operations/query.md)** - Fetch records with filtering
- **[QueryAll](/docs/operations/queryall.md)** - Fetch all records
- **[QueryMultiple](/docs/operations/querymultiple.md)** - Execute multiple queries in one call

## Insert Operations

- **[Insert](/docs/operations/insert.md)** - Create a single record
- **[InsertAll](/docs/operations/insertall.md)** - Create multiple records (batch)
- **[BulkInsert](/docs/operations/bulkinsert.md)** - High-performance insert

## Update Operations

- **[Update](/docs/operations/update.md)** - Update a single record
- **[UpdateAll](/docs/operations/updateall.md)** - Update multiple records (batch)
- **[BulkUpdate](/docs/operations/bulkupdate.md)** - High-performance update

## Delete Operations

- **[Delete](/docs/operations/delete.md)** - Delete a single record
- **[DeleteAll](/docs/operations/deleteall.md)** - Delete multiple records (batch)
- **[BulkDelete](/docs/operations/bulkdelete.md)** - High-performance delete
- **[Truncate](/docs/operations/truncate.md)** - Clear all records

## Merge Operations

- **[Merge](/docs/operations/merge.md)** - Insert or update based on qualifiers
- **[MergeAll](/docs/operations/mergeall.md)** - Batch merge operation
- **[BulkMerge](/docs/operations/bulkmerge.md)** - High-performance merge

## Aggregate Operations

- **[Average](/docs/operations/average.md)** - Calculate average value
- **[AverageAll](/docs/operations/averageall.md)** - Average for all records
- **[Count](/docs/operations/count.md)** - Count records matching criteria
- **[CountAll](/docs/operations/countall.md)** - Count all records
- **[Max](/docs/operations/max.md)** - Get maximum value
- **[MaxAll](/docs/operations/maxall.md)** - Maximum value for all records
- **[Min](/docs/operations/min.md)** - Get minimum value
- **[MinAll](/docs/operations/minall.md)** - Minimum value for all records
- **[Sum](/docs/operations/sum.md)** - Sum values
- **[SumAll](/docs/operations/sumall.md)** - Sum all values
- **[Exists](/docs/operations/exists.md)** - Check if record exists

## Batch Query Operations

- **[BatchQuery](/docs/operations/batchquery.md)** - Execute batch queries

## Direct Execution

- **[ExecuteQuery](/docs/operations/executequery.md)** - Execute raw query
- **[ExecuteNonQuery](/docs/operations/executenonquery.md)** - Execute command without result
- **[ExecuteScalar](/docs/operations/executescalar.md)** - Execute and return single value
- **[ExecuteReader](/docs/operations/executereader.md)** - Execute and get data reader
- **[ExecuteQueryMultiple](/docs/operations/executequerymultiple.md)** - Execute multiple queries

## PostgreSQL and SqlServer -Specific Operations

- **[BinaryImport](/docs/operations/binaryimport.md)** - Binary data import (PostgreSQL)
- **[BinaryBulkInsert](/docs/operations/binarybulkinsert.md)** - Binary bulk insert (PostgreSQL)
- **[BinaryBulkDelete](/docs/operations/binarybulkdelete.md)** - Binary bulk delete (PostgreSQL)
- **[BinaryBulkDeleteByKey](/docs/operations/binarybulkdeletebykey.md)** - Binary bulk delete by key (PostgreSQL)
- **[BinaryBulkMerge](/docs/operations/binarybulkmerge.md)** - Binary bulk merge (PostgreSQL)
- **[BinaryBulkUpdate](/docs/operations/binarybulkupdate.md)** - Binary bulk update (PostgreSQL)
