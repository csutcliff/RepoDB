# Mappers

Fluent mappers and configuration utilities for advanced customization.

## Main Mapper

- **[FluentMapper](/docs/mappers/FluentMapper.md)** - Main fluent API for entity configuration

## Specialized Mappers

- **[ClassMapper](/docs/mappers/classmapper.md)** - Map classes to table names
- **[PropertyMapper](/docs/mappers/propertymapper.md)** - Map properties to columns
- **[ClassHandlerMapper](/docs/mappers/classhandlermapper.md)** - Map class handlers
- **[PropertyHandlerMapper](/docs/mappers/propertyhandlermapper.md)** - Map property handlers
- **[PropertyValueAttributeMapper](/docs/mappers/propertyvalueattributemapper.md)** - Map property value attributes

## Database Configuration

- **[DbSettingMapper](/docs/mappers/dbsettingmapper.md)** - Map database settings
- **[DbHelperMapper](/docs/mappers/dbhelpermapper.md)** - Map database helpers
- **[StatementBuilderMapper](/docs/mappers/statementbuildermapper.md)** - Map statement builders

## Key Mapping

- **[PrimaryMapper](/docs/mappers/primarymapper.md)** - Map primary keys
- **[IdentityMapper](/docs/mappers/identitymaper.md)** - Map identity columns

## Type Configuration

- **[TypeMapper](/docs/mappers/typemapper.md)** - Map data types
